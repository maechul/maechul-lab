# 매출연구소 디자인 시스템 (Design System)

> 목적
>
> 매출연구소 홈페이지의 모든 디자인은
> 아래 규칙을 기준으로 제작한다.
>
> 목적은 "예쁜 디자인"이 아니라
> "신뢰를 만들고 계약을 만드는 디자인"이다.

---

# 1. 브랜드 철학

매출연구소는

영상 편집 업체가 아니다.

사장님의 매출을 연구하는 브랜드다.

디자인 또한

화려함보다

신뢰감을 우선한다.

---

# 2. 디자인 키워드

- Premium
- Trust
- Simple
- Modern
- Professional
- Minimal

---

# 3. 전체 분위기

과한 애니메이션을 사용하지 않는다.

과한 색상을 사용하지 않는다.

불필요한 장식을 넣지 않는다.

여백을 적극적으로 활용한다.

첫인상은 조용하지만

전문성이 느껴져야 한다.

---

# 4. 컬러 시스템

텍스트

White

포인트 컬러

Brand Blue

브랜드 블루는

사이트 전체를 파랗게 만드는 것이 아니라

강조 요소에만 사용한다.

사용 위치

- Active Menu
- CTA Hover
- Link Hover
- Icon
- Focus
- 강조 텍스트

## Background Strategy

Intro와 Hero는 브랜드의 첫인상을 위해 Dark Theme를 사용한다.

본문 콘텐츠는 가독성을 최우선으로 하며, White 또는 Light Theme를 기본으로 사용한다.

Footer는 Dark Theme를 사용하여 사이트를 안정감 있게 마무리한다.

---

# 5. 타이포그래피

제목은

짧고 강하게.

본문은

읽기 쉽게.

한 문장은

가능하면 짧게 작성한다.

줄간격은 넉넉하게 유지한다.

---

# 6. 레이아웃

Desktop

최대 폭을 유지한다.

좌우 여백을 충분히 확보한다.

Mobile

세로 흐름을 우선한다.

터치하기 쉽게 만든다.

---

# 7. Hero

역할

브랜드의 첫인상

Hero에서는

과한 설명을 하지 않는다.

버튼을 최소화한다.

배경은

영상 또는 이미지

텍스트는

2줄 이내

---

# 8. Header

Hero에서는 숨김

첫 스크롤부터 등장

한 번 등장하면

페이지 최상단으로 이동해도 유지

Sticky Header 사용

현재 Section은 Active 표시

---

# 9. CTA

CTA는

신뢰가 형성된 이후 등장한다.

버튼은

크고 누르기 쉬워야 한다.

40~60대 사용자를 고려한다.

---

# 10. 버튼

둥근 모서리

충분한 높이

넓은 클릭 영역

Hover는 절제한다.

튀는 효과는 사용하지 않는다.

---

# 11. 아이콘

라인 아이콘 위주

단순한 스타일

필요한 곳만 사용

---

# 12. 애니메이션

Fade

Slide

Opacity

위주만 사용

Bounce

Zoom

과한 Motion 효과는 사용하지 않는다.

---

# 13. 이미지 / 영상

Hero

브랜드 분위기

Brand Section

Loop Video

Portfolio

실제 작업 영상

이미지는

고화질만 사용한다.

---

# 14. 여백

여백은 디자인이다.

답답한 화면을 만들지 않는다.

콘텐츠보다

여백을 먼저 생각한다.

---

# 15. UX 원칙

사용자가

생각하지 않아도

사용할 수 있어야 한다.

복잡한 UI는 만들지 않는다.

직관성을 최우선으로 한다.

---

# 16. 개발 원칙

디자인보다

유지보수를 우선한다.

HTML은 Semantic하게 작성한다.

CSS는 확장 가능하게 작성한다.

JavaScript는 필요한 만큼만 작성한다.

---

# 17. 최종 목표

이 홈페이지는

포트폴리오 사이트가 아니다.

계약을 만드는 영업 도구다.

모든 기능은

"계약 가능성을 높이는가?"

를 기준으로 추가한다.

# 18. 삭제의 원칙

기능을 추가하기 전에

먼저 제거할 수 있는지 고민한다.

같은 목적이라면

더 단순한 UI를 선택한다.

새로운 기능은

계약 가능성을 높일 때만 추가한다.

예쁘기 때문에 넣는 기능은 만들지 않는다.
